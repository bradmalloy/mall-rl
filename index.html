Info

Welcome to my tiny roguelike!

Direction keys (clockwise) for all 8 directions are: W, E, D, C, X, Z, A, Q. To wait for a turn, press: S. Pickup items ("*") with G. Enemy thugs are represented by the character 't'.

I made this to learn how to use rot.js, an absolutely delightful Javascript library for writing roguelikes. If you're at all interested in learning to make roguelikes, rot.js provides loads of utilities, but isn't opinionated about how you use them. I'm also using NES.css for the styling.
